Xmldiff

Xmldiff is an OCaml library to compute diffs on XML trees.

Download

Xmldiff is hosted on Github.

Releases
  • 0.2 [2014-09-08]:
    • Fix #2: error when comparing <foo/> to <foo><bar/></foo>,
    • use polymorphic variants not to require -rectypes flag,
    • compile and install xmldiff.cmti,
    • new parameter ~cut for Xmldiff.diff to quicken the comparison by lowering precision under some tags.
  • 0.1 [2014-03-03]: Initial release.
Installation

Xmldiff requires Xmlm >= 1.1.0 to be installed.

To compile and install:

  • tar xvfz xmldiff-XXX.tar.gz
  • cd xmldiff-XXX
  • make all install

This will install the xmldiff package with findlib.
