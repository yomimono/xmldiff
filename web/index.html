Xmldiff is an OCaml library to compute diffs on XML trees.

Xmldiff is hosted on Github.

  • 0.1 [2014-03-03]: Initial release.

Xmldiff requires Xmlm >= 1.1.0 to be installed.

To compile and install:

  • tar xvfz xmldiff-XXX.tar.gz
  • cd xmldiff-XXX
  • make all install

This will install the xmldiff package with findlib.
